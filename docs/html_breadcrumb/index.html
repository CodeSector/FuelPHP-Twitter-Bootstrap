Html Breadcrumb plugin

breadcrumb( array(items) array($attrs) )

  1. divider => (string) default: '/' or string(divider): item's separator
  2. Item element: array( 'anchor', 'text', (array)attrs)
  3. Item attributes supports icon key.
Html::breadcrumb(array(
	array('#', 'Home', array('icon' => 'home')),
	array('#', 'Categories'),
	'Bootstrap'
))
  • Home/
  • Categories/
  • Bootstrap

add( anchor, text, array($attrs) ))

Html::breadcrumb(array('divider' => ' - '))
	->add('#', 'home', array('icon' => 'home'))
	->add('#', 'categories')
	->add('#', 'breadcrumb');

render()

returns the Breadcrumb

echo $breadcrumb->render();
