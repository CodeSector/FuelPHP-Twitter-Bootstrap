Progress bars See documentation

Default progress bar, where the value is the % of the progression: 
Html::progress(64)
The status can be speficied as string, or by a key if you need to add classic html attributes: 
Html::progress(36, 'success')
// or
Html::progress(36, array('status' => 'success'))
success
info
warning
danger
The third parameter let's you the ability to add some text whithin the bar: 
Html::progress(46, 'info', '56 sec remaining')
56 sec remaining
The striped attribute generates a striped effect: 
Html::progress(36, array('striped' => true))
The active, in conjonction with striped, animates the stripes 
Html::progress(36, array('striped' => true, 'active' => true))

Stacked progress bar

Add multiple bars into the same progress instance 
pro = Html::progress();
$pro->bar(15);
$pro->bar(15, 'info')->tooltip('Second Bar');
$pro->bar(15, 'warning', 'End');
End
Setup attributes for all bars, and add text whithin each: 
$pro = Html::progress(array('active' => true, 'striped' => true));
$pro->bar(30, 'success', 'This');
$pro->bar(30, 'warning', 'is');
$pro->bar(30, 'danger', 'Reggae');
